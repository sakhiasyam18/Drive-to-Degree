## 🎮 Tentang Game Ini
**Drive to Degree** adalah gim simulasi kehidupan kampus 3D dengan elemen RPG (*Role-Playing Game*) yang dikembangkan sebagai Tugas Akhir Pengembangan Gim Semester 4. Pemain akan berperan sebagai mahasiswa Universitas Negeri Malang yang harus menyeimbangkan eksplorasi dunia kampus, mematuhi etika dan aturan lalu lintas, serta berinteraksi dengan civitas akademika untuk mencapai kelulusan.

### ✨ Fitur Utama
* **🏍️ Eksplorasi Kampus Dinamis:** Susuri setiap sudut lingkungan kampus *open-world* dengan mekanik berkendara motor yang responsif.
* **🗣️ Sistem Dialog Interaktif (Custom Reply):** Berkomunikasi dua arah dengan Dosen dan mahasiswa lain melalui UI dialog yang imersif.
* **🧠 Kecerdasan Buatan (NavMesh AI):** NPC (Karakter Non-Pemain) yang hidup, mampu berjalan secara dinamis, menghindari rintangan, dan merespons interaksi pemain.
* **📚 Simulasi Akademik Nyata:** Kumpulkan Skor Etika, hadiri kelas, dan buktikan kamu layak meraih gelar sarjana!
